# Simple adding of user record

This is a test project for BME.
You should have Node.JS installed in able to test

# How to use

1. Extract zip file to a specific folder
2. Open the folder location using vscode or any code editor
3. On the code editor, open terminal and run:
> npm install
4. Once done, on the same terminal run:
> npm run dev
5. Open the url as given, usually on https://localhost:3000
